
# 硅谷之窗 MVP

这是一个 Next.js 虚拟产品商城原型，适合零基础先跑通产品形态。

## 已包含功能

- 首页
- 虚拟产品商城
- 商品详情页
- 模拟购买
- 我的订单 / 下载中心
- 管理员上传商品页面原型

## 本地运行

1. 安装 Node.js：建议 LTS 版本
2. 解压本项目
3. 在项目目录运行：

```bash
npm install
npm run dev
```

4. 浏览器打开：

```bash
http://localhost:3000
```

## 下一步建议

- 接入 Supabase：用户注册登录、商品数据库、订单表、文件存储
- 接入支付：Stripe / Lemon Squeezy / Paddle 或国内支付方案
- 加入管理员权限
- 商品上传后自动展示在商城
- 支付成功后自动解锁下载权限
